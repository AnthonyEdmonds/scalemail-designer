Scale Mail Inlay Designer

Created by Anthony Edmonds | v1.3 | 15/1/2016

You're going to need JavaScript to use this inlay designer, sorry.

Features

  • Customisable pattern size
  • Estimated size and weight (M/I)
  • Recolour individual scales
  • Save and Load your inlay designs
  • Scales per colour breakdown

Suggestions

  • Get real images of: Blue, Bronze, Champagne, Gold (Shiny)
  • Paintbrush mode
  • Insert Row/Column
  • Copy Row/Column Pattern
  • Server Gallery

Tips

  • Colour Name: Hover over a scale in the palette to see its full name
  • Select: Click the bottom half of a scale
  • Zoom: Ctrl + Scroll Wheel
  • Zoom Reset: Ctrl + 0
